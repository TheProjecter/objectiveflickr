#defines


OFAuthenticationEndPointKey

#define OFAuthenticationEndPointKey @"authEndPoint"
Discussion

The key for Flickr's authentication service end point. You have to open the user's default browser and points it to the URL associated with this key (i.e. the key-value of the API end point dictionary) to complete the Flickr authentication.

OFPhotoURLPrefixKey

#define OFPhotoURLPrefixKey @"photoURLPrefix"
Discussion

The prefix of Flickr's static photo URL.

OFRESTAPIEndPointKey

#define OFRESTAPIEndPointKey @"RESTAPIEndPoint"
Discussion

The key for Flickr REST API end point.

OFUploadCallBackEndPointKey

#define OFUploadCallBackEndPointKey @"uploadCallBackEndPoint"
Discussion

The callback URL to which you can point your browser after a picture is successfully uploaded.

OFUploadEndPointKey

#define OFUploadEndPointKey @"uploadEndPoint"
Discussion

The key of the end point of Flickr's upload service.
