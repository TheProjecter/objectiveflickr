Shared Error Codes

enum { 
    OFConnectionError = -1, 
    OFConnectionTimeout = -2, 
    OFConnectionCanceled = -3, 
    OFXMLDocumentMalformed = -4, 
};
Constants
OFConnectionError
Indicates a system-wide connection error has occured. Check the errorInfo/errorCode object passed to your delegate or callback method for detailed diagnosis.
OFConnectionTimeout
The request has timed out. The timeout interval is usually set when you create an ObjectiveFlickr request object.
OFConnectionCanceled
The request has been explicitly canceled.
OFXMLDocumentMalformed
The request is completed, but the received data may be corrupt or malformed that it cannot be converted to an XML document object.
Discussion

There are the error codes used framework-wide by ObjectiveFlickr objects
